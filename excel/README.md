# 本地资金流向链条分析工具

这是一个完全本地运行的小工具，不联网，不上传数据。它读取 `.xlsx` 或 `.csv` 交易流水，自动寻找：

> 前一笔的汇入方/收款人，在同一天或之后 N 天内，又作为汇出方/付款人转给别人。

## 启动

双击：

```text
启动资金流向分析工具.bat
```

或在当前目录运行：

```powershell
python fund_flow_app.py
```

如果用户电脑没有 Python，源码版不能直接运行。给完全不懂代码的用户使用时，推荐先在开发电脑双击：

```text
打包成EXE_开发者用.bat
```

打包完成后，把 `dist\资金流向分析工具.exe` 发给用户即可。exe 版本不需要用户安装 Python、第三方库，也不需要配置清华源或其他 pip 镜像源。

当前程序只使用 Python 标准库，不依赖 pandas/openpyxl 等第三方库；清华源只在开发者打包时自动安装 PyInstaller 失败或重试时可能相关，普通用户不会用到。

## 输入表字段

至少需要这些列：

- 日期
- 汇出方/付款人
- 汇入方/收款人
- 金额

软件会按列名自动猜测字段，也可以在界面里手动选择。支持 `.xlsx` 和 `.csv`，其中 `.xlsx` 读取功能使用 Python 标准库实现，不依赖 pandas/openpyxl。

## 链条规则

- 相邻天数：默认 `2`，表示同一天、后 1 天、后 2 天都算相邻。
- 最大链长：默认 `5`，避免非常大的表产生过长链条。
- 金额近似容差 %：留空表示不按金额过滤；填 `5` 表示后一笔金额必须和前一笔金额相差不超过 5%。

## 输出

分析后会在原 Excel 所在目录导出两个 CSV：

- `资金流向相邻关系_时间戳.csv`
- `资金流向链条_时间戳.csv`

CSV 用 UTF-8 BOM 编码，Excel 可以直接打开。
